สรุปเศรษฐศาสตร์: กลไกตลาด

บทที่ 3 - อุปสงค์ อุปทาน และความยืดหยุ่น

1. อุปสงค์ (Demand)

ความหมายของอุปสงค์

อุปสงค์ คือ ความต้องการและความสามารถในการซื้อสินค้าหรือบริการในราคาต่างๆ ในช่วงเวลาหนึ่ง โดยสิ่งอื่นคงที่

กฎของอุปสงค์ (Law of Demand)

เมื่อราคาสินค้าเพิ่มขึ้น ปริมาณอุปสงค์จะลดลง และเมื่อราคาสินค้าลดลง ปริมาณอุปสงค์จะเพิ่มขึ้น โดยสิ่งอื่นคงที่

ตัวอย่างการวิเคราะห์การเปลี่ยนแปลงอุปสงค์

  • นิสิตได้ค่าขนมเพิ่มขึ้น → อุปสงค์ดูหนัง เพิ่มขึ้น
  • ราคาโค้กลดลง → อุปสงค์เป๊บซี่ ลดลง (สินค้าทดแทน)
  • ราคาเนื้อหมูเพิ่มขึ้น → อุปสงค์ไก่ เพิ่มขึ้น (สินค้าทดแทน)
  • ราคาน้ำมันเพิ่มขึ้น → อุปสงค์รถยนต์ ลดลง (สินค้าเสริม)

2. ค่าความยืดหยุ่น (Elasticity)

ความหมายของความยืดหยุ่น

ความยืดหยุ่น คือ ค่าที่ใช้วัดความมากน้อยของการเปลี่ยนแปลงของตัวแปรตามต่อการเปลี่ยนแปลงของตัวแปรต้น

สูตรคำนวณความยืดหยุ่น:
ความยืดหยุ่น = % การเปลี่ยนแปลงของตัวแปรตาม / % การเปลี่ยนแปลงของตัวแปรต้น

ความยืดหยุ่นของอุปสงค์ต่อราคา

สูตรแบบจุด:
Ed = (ΔQ/Q) / (ΔP/P)

สูตรแบบช่วง:
Ed = [(Q2-Q1)/(Q2+Q1)/2] / [(P2-P1)/(P2+P1)/2]

ระดับความยืดหยุ่น

ค่าความยืดหยุ่น ประเภท ความหมาย
= 0 ไม่มีความยืดหยุ่น ราคาเปลี่ยน ปริมาณไม่เปลี่ยน
0 < |Ed| < 1 ความยืดหยุ่นน้อย ราคาเปลี่ยน 1% ปริมาณเปลี่ยนน้อยกว่า 1%
|Ed| = 1 ความยืดหยุ่นหน่วย ราคาเปลี่ยน 1% ปริมาณเปลี่ยน 1%
|Ed| > 1 ความยืดหยุ่นมาก ราคาเปลี่ยน 1% ปริมาณเปลี่ยนมากกว่า 1%
ตัวอย่างโจทย์:
ช็อคโกแล็ตราคาเดิม 25 บาท ขายได้ 1,000 แท่ง
ราคาลดเหลือ 13 บาท ขายได้ 1,500 แท่ง

วิธีทำ:
Ed = [(1500-1000)/(1500+1000)/2] / [(13-25)/(13+25)/2]
Ed = [500/1250] / [-12/19] = 0.4 / (-0.63) = -0.63

ความยืดหยุ่นไขว้

  • ค่าบวก (+): สินค้าทดแทนกัน (Substitute)
  • ค่าลบ (-): สินค้าเสริมกัน (Complement)

ความยืดหยุ่นต่อรายได้

  • ค่าบวก (+): สินค้าปกติ (Normal Goods)
  • ค่าลบ (-): สินค้าด้อยคุณภาพ (Inferior Goods)

3. อุปทาน (Supply)

ความหมายของอุปทาน

อุปทาน คือ ปริมาณสินค้าหรือบริการที่ผู้ขายยินดีและสามารถเสนอขายในราคาต่างๆ ในช่วงเวลาหนึ่ง โดยสิ่งอื่นคงที่

กฎของอุปทาน (Law of Supply)

เมื่อราคาสินค้าเพิ่มขึ้น ปริมาณอุปทานจะเพิ่มขึ้น และเมื่อราคาสินค้าลดลง ปริมาณอุปทานจะลดลง โดยสิ่งอื่นคงที่

ปัจจัยกำหนดอุปทาน

  • ต้นทุนการผลิต: ค่าแรง วัตถุดิบ เทคโนโลยี
  • จำนวนผู้ขาย: ผู้ขายเพิ่ม อุปทานเพิ่ม
  • นโยบายรัฐ: ภาษี เงินอุดหนุน
  • สภาพธรรมชาติ: ภัยแล้ง อุทกภัย
  • ความคาดหวัง: คาดการณ์ราคาในอนาคต

4. ดุลยภาพตลาด (Market Equilibrium)

ดุลยภาพตลาด เกิดขึ้นเมื่อปริมาณอุปสงค์เท่ากับปริมาณอุปทาน ณ ราคาหนึ่ง

ราคาดุลยภาพ: ราคาที่ตลาดสมดุล

ปริมาณดุลยภาพ: ปริมาณที่ซื้อขายกันจริงในตลาด

การเปลี่ยนแปลงดุลยภาพ

  • อุปสงค์เพิ่ม: ราคาขึ้น ปริมาณขึ้น
  • อุปสงค์ลด: ราคาลง ปริมาณลง
  • อุปทานเพิ่ม: ราคาลง ปริมาณขึ้น
  • อุปทานลด: ราคาขึ้น ปริมาณลง

5. การแทรกแซงราคาโดยรัฐ

ราคาขั้นสูง (Price Ceiling)

  • รัฐกำหนดราคาไว้ ต่ำกว่า ราคาตลาด
  • เกิด อุปสงค์ส่วนเกิน (Shortage)
  • ตัวอย่าง: ค่าเช่าคอนโด เจลแอลกอฮอล์

ราคาขั้นต่ำ (Price Floor)

  • รัฐกำหนดราคาไว้ สูงกว่า ราคาตลาด
  • เกิด อุปทานส่วนเกิน (Surplus)
  • ตัวอย่าง: ค่าแรงขั้นต่ำ การประกันราคาข้าว

สรุปสำคัญ

การเข้าใจกลไกตลาดจะช่วยให้เราวิเคราะห์การเปลี่ยนแปลงราคาและปริมาณในตลาดต่างๆ ได้อย่างถูกต้อง และสามารถคาดการณ์ผลกระทบจากเหตุการณ์ต่างๆ ได้
